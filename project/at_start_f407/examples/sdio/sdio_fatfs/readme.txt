  this demo is based on the at-start board, in this demo, sdio1 to manage
  test FatFs R0.14b.
  
    sdio1                                           sd/mmc card
  - sdio1_d0                    pc8          --->   dat0
  - sdio1_d1                    pc9          --->   dat1
  - sdio1_d2                    pc10         --->   dat2
  - sdio1_d3                    pc11         --->   dat3
  - sdio1_ck                    pc12         --->   clk
  - sdio1_cmd                   pd2          --->   cmd
  for more detailed information. please refer to the application note document AN0105.
